Class Projects

Here are some of the class projects that I have done at UCI. For academic honesty reasons, I cannot post a GitHub link.


Hash Table (C++): Wrote an authentication service that acted as a hash map from scratch. It kept track of username/password combinations, ensured those combinations are valid, and reported the on number of unique combinations. To operate, it read input from stdin and provided error checking to make sure the user did not input anything incorrectly. For the buckets, I implemented a linked-list. Additionally, we were also tasked with effectively managing dynamically allocated memory.

BooEdit Text Editor (C++): Created a basic text editor using the command pattern while being provided the editor model by the professor. Implemented the use of inheritance and a stack to keep track of user commands so we could undo/redo them. Some commands that were implemented include: cursor up, delete line, undo, redo, new line etc.

Mapquest API (Python): Utilized Mapquest API in Python to build a routing service providing directions, elections, latitude and longitude, time of the trip, and distance of directions to multiple locations anywhere in the world using an intuitive UI.

Board Game Engine (Java): Created a Java game driver and engine using polymorphism that supports two-player board games. Displayed an interface that is flexible to handle games such as Tic-Tac-Toe, Connect Four, Chess, and Checkers.
